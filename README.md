# Two_axis_machine_STM32_Control
Course project that used an STM32, with a stepper motor driver, 2 stepper motors, 2 potentiometers, limit switches, and an LED to accurately control the position of a slider on a 2-axis machine by controlling the speed, direction and motion of motors.
